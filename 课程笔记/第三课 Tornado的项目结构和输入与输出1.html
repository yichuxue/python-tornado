1、通过get_argument（）函数获取查询字符串或者请求体中的信息

查询字符串：就是在浏览器的地址栏看到的url中“？”后面的字符串
请求体：在POST请求中的body数据，比如表单数据form data、json数据、xml数据

http://www.baidu.com/s? wd=Tornado&rsv_spt=1

获取名称为hello的数据
class IndexHandler(tornado.web.RequestHandler):
    def get(self):
        '''通过get传入参数'''
        name = self.get_argument('hello', 'no')
        self.write(name)


函数说明

函数原型：（返回具有给定名称的参数的值。返回的值始终是unicode）
RequestHandler.get_argument（name，default = <object object>，strip = True ）[source]

参数说明：

- 如果未提供default 值，则认为该参数是必需的，如果缺少该参数则抛出 MissingArgumentError异常。

- 如果参数多次出现在url中，我们返回最后一个值。

- strip = True，默认为剪切字符串两边的空格

4、通过URl获取参数（可以使用REST风格url）

REST风格url对比
查询字符串风格
http://1921.168.201.130:8000/book?name=world&num=3

符合REST风格url
http://192.168.201.130:8000/book/world/3

在路由映射中的写法
(r'/book/(.+)/([1-9]+)', BookHandler),#通过url名称不固定的传参

(r'/bookname/(?P<name>.+)/(?P<age>[1-9]+)', BookNameHandler),#通过url传固定名称的参数,名字固定为name age

handler中的写法

class BookHandler(tornado.web.RequestHandler):
    def get(self, name, age):
        '''通过url传入参数'''
        self.write('------name=%s --------age=%s-----------'%(name,age))

class BookNameHandler(tornado.web.RequestHandler):
    def get(self, name, age):
        '''通过url传入参数名是固定的'''
        print self.request.remote_ip        
        self.write('-----name=%s --------age=%s-----%s------'%(name, age, self.request.remote_ip))

函数原型:

  • 将给定的块写入输出缓冲区。
  • 要将输出写入网络，请使用下面的flush（）方法
  • 如果给定的块是字典，我们将其写为JSON，并将响应的Content-Type设置为application/json。（如果要发送JSON作为其他的Content-Type，调用write（）后调用set_header ）。

7、self.request可获取到的请求信息

method
HTTP请求方法，例如“GET”或“POST”
uri
 请求的完整uri
path
路径部分的uri
query
查询部分的uri
version
请求中指定的HTTP版本，例如“HTTP / 1.1”
headers
HTTPHeaders用于请求标题的类似字典的对象。request.headers["aaa"] 像一个不区分大小写的字典一样使用附加的重复标题的方法。
body
请求主体（如果存在）作为字节串
remote_ip
客户端的IP地址作为字符串。如果HTTPServer.xheaders设置，将传递由负载均衡器
在X-Real-Ip或X-Forwarded-For头中提供的真实IP地址。版本3.1更改：X-Forwarded-For现在支持列表格式。
protocol
所使用的协议是“http”或“https”。如果HTTPServer.xheaders 设置，将通过负载均衡器使用的协议（如果通过X-Scheme头部报告）传递。
host
请求的主机名，通常取自Host标题
arguments
参数属性中提供GET / POST参数，它将参数名称映射到值列表（以支持单个名称的多个值）。名称是类型str，而参数是字节字符串。请注意，这不同于 RequestHandler.get_argument，它将参数值作为unicode字符串返回。
query_arguments
与arguments格式相同 但只包含从查询字符串中提取的参数。版本3.2中的新功能
body_arguments
与arguments格式相同 但只包含从请求体提取的参数。版本3.2中的新功能。
files
文件属性中可以使用文件上传，它将文件名映射到列表HTTPFile。 tornado.httputil.HTTPFile表示通过表单上传的文件。为了向后兼容，其实例属性也可作为字典键访问。
connection
HTTP请求附加到单个HTTP连接，可以通过“连接”属性进行访问。由于连接通常在HTTP / 1.1中保持打开状态，所以可以在单个连接上顺序处理多个请求cookies:Cookie.Morsel对象的字典
full_url()
重新构建此请求的完整URL。
request_time()
返回此请求执行所花费的时间。
